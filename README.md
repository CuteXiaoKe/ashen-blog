# ashen-blog

> ashen's blog, koa 2.x + vue 2.x 

> 我的自建博客，使用koa 2.x + vue 2.x构建


## Build Setup

### Client端

#### dev

```bash
npm run dev-client
```

#### build

```bash
npm run build-client
```

### Admin端

#### dev

```bash
npm run dev-admin
```

#### build

```bash
npm run build-admin
```

### Server端

#### dev

```bash
npm run dev-server
```

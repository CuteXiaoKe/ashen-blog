## Build Setup

### Client端

#### dev

```bash
npm run dev-client
```

#### build

```bash
npm run build-client
```

### Admin端

#### dev

```bash
npm run dev-admin
```

#### build

```bash
npm run build-admin
```

### Server端

#### dev

```bash
npm run dev-server
```
